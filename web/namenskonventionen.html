✏️ Schreibstile im Überblick

camelCase

Camel Case

getUserName()
shoppingCartItems
isLoggedIn

Erstes Wort klein, jedes folgende Wort mit Großbuchstaben. Standard für Variablen und Methoden in JS, Java, Swift u.a.

PascalCase

Pascal Case

UserProfile
ShoppingCart
HttpRequest

Jedes Wort beginnt mit Großbuchstaben. Verwendet für Klassen, Typen, Interfaces und React-Komponenten.

snake_case

Snake Case

user_name
max_retry_count
get_user_by_id

Alle Wörter klein, durch Unterstriche getrennt. Typisch in Python, Ruby, PHP sowie SQL-Spaltenbezeichnungen.

SCREAMING_SNAKE

Screaming Snake Case

MAX_RETRY_COUNT
API_BASE_URL
DEFAULT_TIMEOUT

Wie snake_case, aber vollständig in Großbuchstaben. Universeller Standard für Konstanten und Umgebungsvariablen.

kebab-case

Kebab Case

user-profile
my-component.tsx
font-size: 16px

Wörter durch Bindestriche verbunden, alles klein. Standard in HTML-Attributen, CSS-Eigenschaften, URLs und Dateinamen.

_underscore

Leading Underscore

_privateHelper()
__dunder__
_internal_state

Ein führender Unterstrich signalisiert private/interne Member. Zwei Unterstriche (Dunder) kennzeichnen Python-Spezialattribute.

✅ Do's & Don'ts

✔ GOOD

  • getUserById()
  • isAuthenticated
  • MAX_RETRY_COUNT
  • UserProfileCard
  • product_description
  • handleFormSubmit()
  • BaseRepository

✘ AVOID

  • GetUserById()
  • authenticated_flag
  • maxretrycount
  • userProfileCard
  • ProductDescription
  • handle_form_submit()
  • baserepository
Goldene Regel: Konsistenz im Team schlägt persönliche Vorlieben. Ein Linter (ESLint, Pylint, Checkstyle) sorgt für automatische Durchsetzung.

🌐 Konventionen nach Sprache

🟨 JavaScript / TypeScript

  • Variablen & Funktionen: camelCase
  • Klassen / Interfaces: PascalCase
  • Konstanten: UPPER_SNAKE
  • Dateien: kebab-case.ts

🐍 Python

  • Variablen & Funktionen: snake_case
  • Klassen: PascalCase
  • Konstanten: UPPER_SNAKE
  • Module: snake_case.py

☕ Java

  • Variablen & Methoden: camelCase
  • Klassen: PascalCase
  • Konstanten: UPPER_SNAKE
  • Pakete: lowercase

🦀 Rust

  • Variablen & Funktionen: snake_case
  • Typen & Enums: PascalCase
  • Konstanten: UPPER_SNAKE
  • Module: snake_case

🐹 Go

  • Öffentlich (exported): PascalCase
  • Privat (unexported): camelCase
  • Konstanten: PascalCase
  • Dateien: snake_case.go

💎 Ruby

  • Variablen & Methoden: snake_case
  • Klassen & Module: PascalCase
  • Konstanten: UPPER_SNAKE
  • Symbole: :snake_case

🎨 CSS / SCSS

  • Klassen & IDs: kebab-case
  • BEM-Block: block__element
  • BEM-Modifier: block--modifier
  • Custom Props: --my-color

🗄️ SQL

  • Tabellen: snake_case
  • Spalten: snake_case
  • Keywords: UPPERCASE
  • Aliase: snake_case

📁 Dateien & Ordnerstruktur

kebab-case

Allgemeine Web-Projekte

user-profile.ts
shopping-cart.css
api-client.test.js

Konsistenz mit URLs und HTML-Attributen, plattformunabhängig.

PascalCase

React / Vue Komponenten

UserCard.tsx
ModalDialog.vue
NavBarItem.jsx

Dateiname entspricht dem Komponentennamen — sofort erkennbar.

snake_case

Python-Module

user_service.py
db_connection.py
test_auth.py

PEP 8-Vorgabe: Module und Pakete immer in snake_case benennen.

Tipp: Vermeide Leerzeichen und Sonderzeichen in Dateinamen — sie erzeugen Probleme in Terminals, URLs und CI-Pipelines.

🔀 Git: Branches & Commit-Messages

Branch-Namen

feature/user-auth
fix/login-crash
chore/update-deps
release/v2.4.0

Präfix + kebab-case. Kategorien: feature, fix, chore, docs, refactor

Conventional Commits

feat: add user avatar upload
fix: resolve null pointer in auth
docs: update API readme
refactor!: rewrite cart module

Format: type(scope): beschreibung — ermöglicht automatische Changelogs und Versionierung.

Commit-Typ Bedeutung Beispiel
featNeues Featurefeat: dark mode toggle
fixFehlerbehebungfix: logout redirect loop
docsDokumentationdocs: add setup guide
styleFormatierung, kein Logik-Changestyle: run prettier
refactorRefactoring ohne Bugfix/Featurerefactor: extract auth helper
testTests hinzufügen/änderntest: unit tests for cart
choreBuild, CI, Abhängigkeitenchore: bump eslint to v9
perfPerformance-Verbesserungperf: lazy-load images
